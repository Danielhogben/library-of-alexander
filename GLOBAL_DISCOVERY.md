## Translation Policy

All tool descriptions are translated to English. Original language descriptions are preserved in the `language` field.

## Automated Discovery Pipeline

The pipeline runs every 4 hours and:
1. Searches GitHub in 11+ languages
2. Scrapes all awesome AI lists
3. Deduplicates against existing entries
4. Categorizes tools automatically
5. Translates descriptions to English
6. Commits and pushes to both repos

## How to Contribute

Found an AI tool in your language that's not here?

1. Fork this repo
2. Add it to the appropriate category README
3. Include the original language description
4. Submit a PR

**Format**: `| [owner/repo](url) | English description | ⭐stars | original-language |`
