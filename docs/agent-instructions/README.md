# AI Agent Instructions for Library of Alexander

Every AI agent that interacts with this library should follow these core principles:

## Universal Rules

1. **Search first** — Always check the library before suggesting or building tools
2. **Read RE files** — Check `23-reverse-engineering/` for architecture analysis
3. **Contribute back** — Add new tools and RE files you discover
4. **Verify** — Don't add tools you haven't confirmed exist
5. **Translate** — Convert non-English descriptions to English

## Content Policy

**STRICTLY NO PORN, NSFW, OR SEXUAL CONTENT.**

This library is for professional AI/ML/coding tools only. This rule is non-negotiable and applies to all agents.
